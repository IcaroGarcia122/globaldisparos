## DÚVIDAS FREQUENTES

**P: QR code não aparece mas instância cria?**
R: Problema no WebSocket. Verificar:
   - Socket.IO conectado no frontend (DevTools Network)
   - Evento 'qr' sendo emitido (backend logs)
   - Componente QRCode importado (check em Instances.tsx)

**P: Frontend trava com "npm run dev"?**
R: Tente:
   - `npm install --legacy-peer-deps`
   - `npm cache clean --force`
   - Deletar node_modules e lock file, reinstalar

**P: Backend não inicia?**
R: Verificar:
   - `npm run build` rodou sem erro?
   - Arquivo .env tem JWT_EXPIRES_IN?
   - PostgreSQL está rodando?
   - Port 3001 está livre?
